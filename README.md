# Telegram Autoresponder with Deskgram 2

Telegram Autoresponder is a Deskgram 2 module for automatic replies to incoming Telegram messages. It helps you keep communication alive without manual presence and can combine standard reply rules, exclusions, AI-generated answers, and a background reply layer for broader outreach routes.

## What it can do

- reply automatically to incoming Telegram messages;
- use fixed templates or AI-generated answers;
- exclude unwanted sources or scenarios;
- run as a background communication layer after an outreach launch;
- collect incoming messages for analysis and follow-up;
- apply more advanced reply behavior rules;
- support a larger communication system without constant manual work.

## Quick start

1. Configure the base reply logic.
2. Add exclusions and safety conditions.
3. Enable AI if the replies should be less repetitive.
4. Connect advanced behavior rules if the route needs more control.
5. Assign accounts and start the module.

## Where this module is especially useful

- when incoming replies must not be lost after an outbound campaign;
- when you need a background communication layer instead of constant manual presence;
- when many parallel dialogues make manual reply handling too slow;
- when a standard reply baseline should be combined with AI only where needed.

## Scenario FAQ

### Can I check the module in the browser before installing Deskgram 2?

Yes. The web preview link in this README opens the module directly in the browser, so you can inspect exclusions, AI settings, and reply logic first.

### Is this a standalone module or mostly a support layer for other campaigns?

It can work both ways. On its own, it handles incoming messages automatically. Inside a larger route, it becomes especially useful after outbound messaging or AI outreach.

### When is AI worth enabling here?

AI is worth enabling when you need more natural replies, less repetition, or more flexible handling of incoming messages. If the flow is simple and predictable, rule-based replies can be enough.

### What is the difference between this module and Neuro Chatting?

Autoresponder is more focused on the broader incoming reply layer, while Neuro Chatting is stronger for group and chat engagement scenarios with trigger-based live discussion logic.
